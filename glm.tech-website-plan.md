- **框架**：React 18 + TypeScript

- **动画**：Framer Motion

## 📊 GLM-4.7 核心特性

### **主要能力**
1. **增强性能**：相比前代模型有明显提升
2. **多领域专精**：技术、创意、代码、多语言处理
3. **推理能力**：逻辑推理和问题解决能力增强
4. **上下文理解**：支持更长上下文，保持对话连贯性

- **上下文窗口**：支持128K+ tokens

- **API支持**：提供完整开发接口
